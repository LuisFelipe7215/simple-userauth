# simple-userauth

Esse é um mini projeto feito para testar o Spring Security.

## 🛠️ Como executar 

1. Suba a aplicação Spring boot dentro da pasta raiz usando o comando: 
```bash
./mvnw spring-boot:run
```
2. Entre no endereço http://localhost:8080 no seu navegador.
3. No campo username digite user e no campo password digite password.
4. O user e o password pode ser trocado no WebSecurityConfig.
